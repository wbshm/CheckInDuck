You are helping me build an iOS app MVP.

Project overview:
This is a privacy-first iOS productivity app for daily app-based habits. Users explicitly authorize Screen Time access, select apps using Apple’s Family Controls picker, set a daily deadline, and if a selected app is used for more than 3 minutes in a day, the task is automatically marked as completed. If not completed before the deadline, the app sends reminders.

Tech stack:
- Swift
- SwiftUI
- MVVM
- iOS 16+
- Local-first storage
- Clean, simple architecture
- Avoid overengineering

Product principles:
- Simple and reliable MVP first
- Privacy-first

- We do not inspect content inside third-party apps
